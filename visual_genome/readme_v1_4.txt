# Visual Genome version 1.4 release.

## Changes from pervious versions

- This release contains cleaner object annotations. This is released in `objects.json.zip`.
- The relationships with the new subject and object bounding boxes are released in `relationships.json.zip`.
- We have also released `object_alias.txt` where every line contains synonyms for objects that are annotated with different names in `objects.json.zip`
- We have also released `relationship_alias.txt` where every line contains synonyms for relationships that are annotated with different predicates in `objects.json.zip`


## What did not change from version 1.2

The images, image_metadata, region descriptions, attributes and visual question answers are left unchanged.
